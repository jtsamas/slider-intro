# slider-intro
Implementation of Intro Slider for Android

## Key architecture apporaces

* Usage of AndroidX architecture components: `LiveData`, `ViewModel`, `ViewModelProviders`, `Transformations`
* State management
* Scalability
* User-friendly animation
